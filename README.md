## Automação Web

Este projeto contém um teste de ponta a ponta (E2E) estruturado com o framework **Cypress** para a validação de navegação.

### O que o primeiro teste faz:
1. **Visita** a página oficial de modalidades esportivas do Comitê Olímpico do Brasil (COB).
2. **Procura** o elemento ou link que contém o texto "Basquete".
3. **Clica** no elemento para navegar até a página dedicada ao esporte.
4. **Valida (Asserção)** se a URL final mudou com sucesso e passou a conter o termo `basquete`.
